### 確認方法

リポジトリクローン後 node_modulesのインストール
npm install
yarn install

確認用ローカルサーバー立ち上げ
npm run dev
yarn dev

### アーキテクト

・MVVMアーキテクチャ
Model: Store内のコード
ViewModel : コンポーネント内のscript
View : コンポーネントのコード
・Vue 3
・Pinia（Store)
・TypeScript
・SCSS
・Vite
・Vue CLI
